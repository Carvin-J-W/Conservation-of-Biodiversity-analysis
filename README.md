# Conservation-of-Biodiversity-analysis
In this repository I undergo a basic data anlysis. The two CSV files that contain our data belongs to USA National Park Services. One CSV contains Species data pertaining to the species; and the other CSV contains Observation data which essentially counts how many times a species was observed in a particular park. 

The centeral goal of this analysis is to hone in on endangerment statuses. How many are endangered? Who/what is endangered? Throughout my code (mostly at the bottom of the Jupyter Notebook) I graph some results to visualize the data. I will let the graphs speak for themselves. Needless to say, you can see the endangerment of species throughout the all the parks and see the ratios of endangered species.  

As mentioned this project was suppoessed to be very basic. With time and thought new questions can arise that we can analze even further to gain even more meaningful insights from the data we already have.
